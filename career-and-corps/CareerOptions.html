Career Options

Here, you can find a few examples of the careers a graduate with a computer science degree can pursue.

Data Scientist

Data scientists utilize statistical and computational methods to derive insights from large amounts of data. The often work with both IT and businesses to solve business-related problems. To be a data scientist, it is required to have a background in statistics, as well as programming languages such as Python and R.

If you want more information about data scientists, click here.

Machine Learning Engineer

Machine learning engineers develop and program machines and systems that can learn and apply knowledge without specific direction. They can program services to automatically identify a user's needs and interests. Additionally, machine learning engineerings can work beyond user applications by helping engineer new pioneering technology such as self-driving cars and voice assistants.

If you want more information about web developers, click here.

Software Engineer

"A software engineer develops various applications that enable users to accomplish tasks on their personal computers and electronic devices. Often, software engineers are employed by software publishers or computer systems design firms."

"Software engineers have extensive knowledge of programming languages, software development, and computer operating systems, and they apply engineering principles to software creation. By applying these engineering principles to every stage of the development process, from requirements analysis to the software process, they can create customized systems for individual clients."

"Software developers use various source debuggers and visual development environments to modify, write, and debug software for client applications."

Web Developer

A web developer designs, creates, and animates websites. They are “programmers who specialize in … the development of World Wide Web applications using a client-server model.” As a web developer, you can either work in the front end, back end, or both of programming (To learn about the differences between front end and back end, click here.) There is a high demand for web developers because mobile devices and ecommerce are becoming more popular.
